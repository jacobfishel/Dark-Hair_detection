Function Purpose
imread(image_path, flag) This function is used to read an image from the specified image_path.

flag: Handles how the image should be read:

    - cv2.IMREAD_COLOR, cv2.IMREAD_GRAYSCALE, cv2.IMREAD_UNCHANGED
imshow(window_name, image) Used to show the image in the window

window_name: str to name the window

image: read image
imwrite(filename, image) Used to save an image

filename: path to save image to including image name and .png

image: image to save
