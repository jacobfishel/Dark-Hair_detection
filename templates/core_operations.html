Function Purpose
imread(image_path, flag) Reads an image from the specified image_path.

flag: Handles how the image should be read:

    - cv2.IMREAD_COLOR, cv2.IMREAD_GRAYSCALE, cv2.IMREAD_UNCHANGED
imshow(window_name, image) Shows the image in the window

window_name: str to name the window

image: read image
imwrite(filename, image) Saves an image

filename: path to save image to including image name and .png

image: image to save
